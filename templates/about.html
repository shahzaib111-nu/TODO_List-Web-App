About This App

Welcome to myTODO — a minimal and powerful task manager built with Flask and Bootstrap 5.

✨ Features:
  • Create, edit, and delete your tasks easily
  • Track when each task was added
  • Simple, fast, and mobile-friendly interface
  • Built using Python, Flask, SQLAlchemy, and Bootstrap
📌 Purpose:

This project is designed for learning web development using Flask and for organizing your personal or academic tasks efficiently.
